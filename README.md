# 🌿 웰니스 체크리스트

나를 이해하고, 나답게 살아가기 위한 웰니스 자가 점검 앱

## 로컬 실행

```bash
npm install
npm run dev
```

## GitHub + Vercel 배포 방법

1. GitHub에 새 Repository 생성
2. 이 폴더 파일 전체 업로드 (또는 git push)
3. [vercel.com](https://vercel.com) 접속 → GitHub 연결
4. 해당 Repository Import → **Deploy** 클릭
5. 완료! 자동으로 URL이 생성됩니다

## 기술 스택

- React 18
- Vite 5
- 순수 인라인 스타일 (CSS 라이브러리 없음)
